## Problem Overview
Level 1 (the idiot test)

This level is what we call "The Idiot Test", if you can't complete it, don't give up on learning all you can, but, don't go begging to someone else for the answer, thats one way to get you hated/made fun of. Enter the password and you can continue.

## Penjelasan
Pada soal, kita diminta untuk memasukkan suatu password. Terdapat clue pada soal, yaitu kata HTML yang sangat menonjol, sehingga mengindikasikan bahwa password yang diminta kemungkinan besar berada pada source code HTML pada website tersebut. Langkah pertama yang perlu kita lakukan adalah mengecek source code website tersebut dengan melakukan Ctrl + Shift + C. Pada tab elements, lakukan analisis secara keseluruhan untuk mencari adanya informasi password. Atau jika ingin lebih cepat, gunakan shortcut Ctrl + F dan masukkan kata kunci Password.
